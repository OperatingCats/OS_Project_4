# OS_Project_4 — Multi-process Blockchain

Exam project: simulated blockchain with Bootstrap, Node, Miner, and Client
processes communicating over Unix-domain sockets, with validation and
mining logic shared between a C implementation and a Bash implementation.

Initial skeleton: source files are empty/placeholders and will be
developed progressively.

## Structure

- [report.md](report.md) — report draft (to be converted to `report.pdf`)
- [code/](code/) — all source code and the build system
  - [code/README.md](code/README.md) — technical instructions
  - [code/Makefile](code/Makefile) — `build`, `run`, `clean`, `test`, etc.
